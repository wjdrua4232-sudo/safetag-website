SafeTag는
산업현장의 안전을 바꿉니다

SafeTag는 UWB(초광대역) 기술을 활용하여 산업현장에서 발생하는 안전 사고를 실시간으로 감지하고 대응하는 통합 안전 플랫폼입니다.

기존 GPS가 불통인 실내·지하 환경에서도 10cm 단위의 정밀 위치 추적이 가능하며, 낙상 자동 감지, 외국인 작업자 다국어 통역, 7단계 긴급구조 프로세스를 하나의 앱으로 제공합니다.

전기·정보통신 전문 기업으로 설립되어 중소 제조 현장을 우선 타깃으로 시장에 진입합니다. 기존 대기업 솔루션 대비 70% 비용 절감으로 중소현장도 부담 없이 도입할 수 있습니다.

SafeTag 핵심 기능

UWB 기술을 기반으로 산업현장 안전에 필요한 모든 기능을 하나의 플랫폼으로 제공합니다.

📡
실시간 위치 모니터링
UWB 앵커와 웨어러블 태그로 현장 작업자 전원의 위치를 10cm 단위로 실시간 파악합니다. 2.5D 등각뷰 디지털트윈 관제 대시보드로 층별·구역별 상세 위치를 한눈에 확인합니다.
정확도 10cm
🆘
낙상 자동 감지 & 긴급 SOS
6축 IMU 센서(ST LSM6DSO)가 낙상 패턴을 실시간 감지하여 사고 즉시 관리자와 구조대에 자동 알림을 전송합니다. SOS 발생 시 혈액형·기저질환 의료정보도 즉시 제공됩니다.
즉시 알림
🌐
외국인 실시간 다국어 통역
한국어·베트남어·중국어·영어 실시간 안전 지시 통역 기능을 제공합니다. 위험 상황에서 언어 장벽 없이 모든 작업자에게 즉각적으로 지시를 전달할 수 있습니다.
4개국어 지원
📱
7단계 긴급구조 프로세스
사고 접수부터 APK 즉각 배포, UWB 정밀 측위, AR 가이드, 신속 구조까지 체계적인 7단계 프로세스로 골든타임을 지킵니다. GPS 불통 환경에서도 벽 너머 위치 파악이 가능합니다.
골든타임 확보

3색 작업자 상태 시스템

직관적인 색상 코드로 현장 전체 상황을 즉시 파악합니다.

GREEN — 일반 작업자
정상 작업 중
지상 작업 중인 일반 작업자. UWB 신호를 정상 송출 중이며 관제 대시보드에서 위치가 실시간 표시됩니다.
YELLOW — 고소작업자
고소작업 모드 활성화
고소작업 모드가 켜진 작업자. 낙상 감지 민감도가 높아지고 관리자 화면에 별도 표시됩니다.
RED — 요구조자
즉각 구조 필요
낙상 감지 또는 SOS 발령. 7단계 긴급구조 프로세스가 자동 시작되고 구조대에 즉시 알림이 전송됩니다.

7단계 프로세스

사고 접수부터 구조 완료까지, 체계적인 프로세스로 골든타임을 지킵니다.

01
사고 접수
실내 고립 및 위치 파악 불가 상황 발생 시 관제 센터 즉시 가동. 낙상 자동 감지 또는 SOS 신호로 트리거됩니다.
관제센터 가동
02
APK 즉각 배포
신고자 및 구조대 휴대폰으로 APK 설치 링크를 문자로 즉각 발송. 설치와 동시에 구조 모드가 활성화됩니다.
문자 발송
03
UWB 신호 활성화
앱 설치 즉시 신고자 기기에서 UWB 신호 송출 시작. GPS 불통 환경에서도 신호가 작동합니다.
신호 송출
04
정밀 측위
구조대 기기가 UWB 신호를 수신하여 센티미터(cm) 단위로 요구조자 위치를 계산합니다.
10cm 정확도
05
AR 가이드
구조대 앱 화면에 AR 화살표로 요구조자 방향을 표시. 연기·어둠 속에서도 화살표를 따라 이동합니다.
AR 화살표
06
장애물 극복
UWB 신호가 콘크리트·벽돌·목재 등 건축 자재를 투과. 벽 너머 위치를 시각적으로 확인하며 이동합니다.
벽 투과 측위
07
신속 구조
직관적인 경로 안내로 골든타임 내 요구조자 발견 및 구조. SOS 시 혈액형·기저질환 정보가 즉시 제공됩니다.
